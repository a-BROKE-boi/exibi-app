# exibi app

## Description

This application retrieves images from the Metropolitan Museum of Art's Open Access API. It allows users to  browse a collection of open-access works from there database gallery, etc.

## Features

*   Retrieves images and metadata from the Met Museum API.

## Technologies Used
*   JavaScript
*   NextJs
*   Tailwind

## Instructions

1.  **Fork this repository:** Click the "Fork" button in the top right corner of this page.
2.  **Clone the forked repository:** Open your terminal and run:

    ```bash
    git clone [invalid URL removed]
    ```

    (Replace `YOUR_USERNAME` with your GitHub username.)

3.  **Navigate to the project directory:**

    ```bash
    cd exibi
    ```

4.  **Install dependencies:** 

    ```bash
    npm install  
    ```

5.  **Obtain an API Key (If required):** The Met Museum's Open Access API doesn't require an API key for basic usage. However, if you are planning on making a high volume of requests, it is advisable to contact them.

6.  **Run the application:** 
    ```bash
    npm run
    ```

7.  **Open the application in your browser:** [`http://localhost:3000`]
